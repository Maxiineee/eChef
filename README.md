Mobile-first webapp to find, create and share recipes. Features illustrated step-by-step instructions and shopping list. Developed with modern technologies granting the best user experience.

Work in progress!!

Stack:
 - Framework: Next.js
 - Language: Typescript
 - Database: Postgres Neon
 - ORM: Prisma
 - Auth: BetterAuth
 - Media Storage: Cloudinary
 - Animations: Framer Motion
 - Styling: Tailwind CSS + Shadcn/UI
 - Deploy: Vercel  

Available at:
echefreceitas.vercel.app
